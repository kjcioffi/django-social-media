# django-social-media

## Time Allocated
* 1 month to complete an MVP

## Tech Stack
* Django
* PostgreSQL
* HTML
* CSS
* Vanilla Javascript

## Requirements
* Create an authentication based social media application.
  * Enable new users to sign up with the platform.
  * A profile is generated with the user account upon sign up / registration.
  * Allow users to update their profiles dynamically.
    * Change profile picture
    * Update bio
* Users must be able to:
  * Create content
  * Interact with content
    * Commenting on posts.
    * Liking posts.
    * Replying to posts
    * Liking replies.
